外贸团队最常收到的问题汇总 — 简洁直答。没有您要的？直接咨询我们。

订单与物流
Q01. 贵司的起订量(MOQ)是多少？
Minimum Order Quantity
+
标准起订量为每款产品200 kg，同批订单支持多色拼单。试订单或新经销商合作，可接受50 kg样品量。定制配方首批生产最少500 kg起。
Q02. 生产和发货周期多久？
Lead Time
+
标品目录产品：7–15个工作日(定金到账起算)。定制配方：30–45天含样品确认。海运整柜至大多数亚洲港口：5–10天。中东/非洲：18–25天。南美/欧洲：28–35天。DHL快递样品：3–5天全球送达。
Q03. 提供哪些包装规格？
Packaging Options
+
5L铁桶(≈5 kg，样品规格) · 20L标准桶(≈25 kg，出口主力) · 200L大桶(≈250 kg，大宗批发) · 1000L IBC吨桶(工业订单)。危化品配方可提供UN认证包装。200kg起可定制自有品牌桶标。
Q04. 支持哪些贸易术语？
Incoterms
+
EXW佛山 · FOB佛山 · CIF目的港 · DDP(选定国家由我方物流伙伴负责)。长期合约保证价格稳定：马士基、地中海、达飞、中远海运。5吨以下订单可LCL拼箱。
Q05. 接受哪些付款方式？
Payment Methods
+
T/T电汇(30%定金 + 70%发货前 — 最常用) · 即期信用证(USD 50,000以上订单) · 长期合作经销商授信账户(6个月以上合作历史)。币种：USD、EUR、RMB。新加坡花旗银行USD账户接收国际电汇。
质量与质保
Q06. 产品有哪些认证？
Certifications
+
ISO 9001:2015(质量管理) · ISO 14001:2015(环境管理) · CE欧盟出口标识 · SGS第三方检测报告 · ASTM E84 A级防火 · 低VOC款 <50 g/L · 应要求提供REACH/RoHS合规 · 饮用水接触NSF/ANSI 61。
Q07. 质保期多久？
Warranty Period
+
产品本身标准24个月质保(自发货日起，针对产品缺陷及批次不一致)。项目性能质保(5–25年，根据体系和环境)可由我方认证施工方现场施工后提供。
Q08. 大货前能先寄样品吗？
Free Samples
+
可以。目录颜色样品(1–2 kg)免费，DHL 48小时内寄出 — 您承担运费(约USD 30–80依目的地)。定制颜色样品：USD 80–150/配方，下大货时可冲抵。
Q09. 如何保证批次一致性？
Batch Consistency
+
每批次均在自有QC实验室进行60+参数测试。批次留样保存24个月。出货前邮件发送COA(检验报告)。关键参数采用SPC统计过程控制。可应要求委托SGS出厂前验货(买方付费)。
定制与OEM
Q10. 可以定制颜色吗？(RAL/Pantone/来样)
Color Matching
+
可以。提供RAL/Pantone编号或≥5 cm²的实物样品。实验室调色3天完成，量产批次7–10天。色差控制：ΔE ≤ 1.0(分光光度计验证)。支持金属、珠光、质感效果。
Q11. 可以定制配方吗？
Custom Formulation
+
可以 — 12位化学工程师研发团队承接定制配方。告知您的基材、环境、性能指标(温度、耐化学、硬度、颜色等)。样品7天，量产30天。保密配方可签NDA及IP协议。
Q12. OEM/贴牌怎么操作？
Private Label
+
从200+既有配方中选择，或定制全新配方。我方按您的品牌生产：定制桶标、品牌包装、多语言MSDS。SKU首批起订500 kg。桶体设计免费(您提供Logo及品牌色)。
技术与施工
Q13. 提供现场施工指导吗？
Site Application Support
+
USD 50,000以上订单或5,000 m²以上项目，我方可派应用工程师赴现场指导：表面处理规范、施工人员培训、QA监督。差旅费按成本收取；服务工时免费。
Q14. 热带气候涂料如何储存？
Storage in Tropical Climates
+
存放于阴凉处，温度5–35°C，避免阳光直射和火源。使用前保持桶盖密封。按FIFO先进先出周转。保质期：12–24个月(依产品)。仓库温度持续超过40°C时建议空调。
Q15. 我的项目该用多厚的涂层？
DFT Selection
+
取决于ISO 12944腐蚀性等级。C2(乡村)：120 μm。C3(城市/轻工业)：160 μm。C4(工业/沿海)：200 μm。C5-I(重工业)：260 μm。C5-M(海洋)：320 μm。告诉我们您的环境，我们给出精确体系。
Q16. 无气喷涂、滚涂还是刷涂？
Application Method
+
无气喷涂：大面积、最快、最佳成膜。中等面积、质感效果、修补用滚涂。边角、小面积、修补用刷涂。宏源产品已优化适配无气喷涂(55:1泵，.017–.023英寸喷嘴)。
