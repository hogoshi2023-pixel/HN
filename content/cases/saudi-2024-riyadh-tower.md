# Project Overview

22,000 m² of exterior facade recoating for a 32-story commercial tower in Riyadh. Project executed during summer months when ambient temperatures exceeded 45°C.

## Solution

Application work was scheduled for night shifts (10pm–6am) to keep substrate temperatures within the 35°C limit. PVDF topcoat system used 70% PVDF resin (vs 50% industry standard), independently verified by SGS for 18-year weathering equivalence.
